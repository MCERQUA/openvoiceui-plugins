# BHB Animated Characters Plugin

Animated character avatars from the **BigHead Billionaires** animated series for OpenVoiceUI.

## What's Included

### Face Avatar
- **BigHead Avatar** — layered character renderer with real-time lip-sync (amplitude-to-viseme), mood-driven eye expressions, and fully customizable appearance
- Assets loaded from the [BHB Character Cooker CDN](https://bhaleyart.github.io/BigHeadCharacterCooker)
- 7 trait categories: Body (33), Head (57), Eyes (29), Mouth (28), Outfit (52), Texture (11), Backgrounds (16)

### Character Builder
- Interactive canvas page for creating and customizing BigHead characters
- 4-step flow: Appearance, Identity (name/personality/backstory), Voice, Review
- Characters saved as agent profiles with face_config embedded

### Default Characters
- **Kyle Valhalla** — the lovable idiot, co-founder of BHB. Chaotic optimist, slow chuckle, unhinged tangents. Resemble AI cloned voice.
- **Bryce BigHead** — Kyle's long-suffering partner. Voice of reason nobody listens to. "God damn it Kyle."

### Show Lore
Full series knowledge for building characters with authentic personalities:
- **Episode transcripts** (Ep 1-4) — raw dialogue from the animated series
- **Character profiles** — Kyle, Bryce (Partner), Axel, Barry G, Reggie, Agent Chill, Tate parody
- **Episode memories** — structured summaries of key events, relationships, consequences
- **World-building** — Block City, Soul Block 420, Utopia, the NFT mint, The Matrix

Agents can access the lore files to build deeper character knowledge, extract memories, or help users create show-accurate characters.
